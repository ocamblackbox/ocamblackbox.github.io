제품 소개

oCamBlackBox의 강력한 기능을 확인하세요

01

오캠블랙박스 소개

오캠블랙박스는 기업의 PC 화면을 실시간으로 녹화하여 데이터 유출, 회계 조작 등의 보안 위협으로부터 기업을 보호하는 통합 보안 솔루션입니다.

Windows 전용 솔루션

  • PC 화면, 프로세스, 브라우저 URL 실시간 녹화
  • USB 복사 및 프린터 출력 파일 추적
  • 키보드, 마우스 이벤트 캡처
  • 지정된 NAS에 안전하게 저장

통합 관리 시스템

  • PC별 녹화 프로그램 설치
  • 중앙 관제 프로그램으로 통합 관리
  • 블랙박스처럼 필요시 검색 및 조회
  • 강력한 암호화 및 보안

검증된 신뢰성

  • 금융기관 및 공공기관 도입 실적
  • 등록 상표 및 다수 특허 보유
  • 안정적인 기술 지원 체계
02

OCAMBB-윈도우

완벽한 화면 녹화

Windows PC의 모든 화면 활동을 빠짐없이 녹화합니다. 프로그램 실행, 마우스 클릭, 키보드 입력까지 모든 활동을 기록합니다.

NAS 통합 저장

모든 녹화 데이터는 지정된 NAS에 안전하게 저장되며, 언제든지 접근 가능합니다.

강력한 암호화

모든 녹화 파일은 암호화되어 저장되며, 전용 뷰어로만 열람 가능합니다.

주요 사양

  • 지원 OS: Windows 8, 10, 11 및 Windows Server 전 버전
  • 모니터: 듀얼 모니터 및 싱글 모니터 녹화 지원
  • 운영 방식: 은폐 모드로 사용자 인지 없이 작동
  • 시스템 부하: 최소한의 리소스 사용으로 PC 성능 영향 최소화
  • 저장 방식: 실시간 압축 및 암호화 저장
05

제품 구성

녹화 프로그램

PC별 라이선스

각 PC에 설치되는 녹화 프로그램입니다.

  • PC당 1개 라이선스 필요
  • 백그라운드 실행
  • 자동 업데이트 지원
  • 중앙 관리 연동

관제 프로그램

1개 라이선스

모든 PC를 관리하는 중앙 관제 프로그램입니다.

  • PC 수량과 무관하게 1개만 필요
  • 전체 PC 통합 관리
  • 녹화 데이터 검색 및 분석
  • 보고서 생성

도입 예시

사무실 PC 20대 운영 시:

  • 녹화 프로그램: 20개 라이선스
  • 관제 프로그램: 1개 라이선스
  • 총 필요 라이선스: 21개

* NAS는 별도 구매 또는 기존 내부 NAS 활용 가능

06

관제프로그램

중앙 관제 시스템의 핵심 기능

모든 PC의 녹화 데이터를 효율적으로 관리하고 분석할 수 있는 통합 관제 프로그램입니다.

녹화 데이터 검색

NAS에 수집된 모든 녹화 데이터를 빠르게 검색하고 조회합니다

사용 통계 분석

프로그램별 사용 시간과 패턴을 분석하여 업무 효율성을 파악합니다

필터링 기능

사용자, 시간대, 실행 프로그램별로 세밀한 필터링이 가능합니다

실시간 모니터링

현재 활동 중인 PC들의 상태를 실시간으로 확인할 수 있습니다

웹사이트 접속 분석

접속한 웹사이트 URL과 체류 시간을 상세히 분석합니다

보고서 생성

다양한 형식의 보고서를 자동으로 생성하여 관리를 용이하게 합니다

녹화 샘플

실제 녹화 화면 예시

oCamBlackBox는 PC 화면의 모든 활동을 선명하게 녹화합니다. 텍스트, 이미지, 동영상 등 모든 화면 요소를 정확하게 캡처하여 필요시 상세한 검토가 가능합니다.

  • 고화질 녹화로 작은 텍스트도 선명하게 확인
  • 마우스 움직임과 클릭 위치 표시
  • 프로그램 전환 및 창 이동 모두 기록
  • 시간별 인덱싱으로 빠른 검색 가능
